# LINKUP_DEV

A modern React + Vite project for a social/community-style web app with authentication, messaging, notifications, and user profile features.

## Features

- User registration and login flow
- Messaging and chat experience
- Notifications and friend/group management
- Responsive UI built with React

## Live Demo

Visit the deployed project here:

- https://linkup-dev-nine.vercel.app/

## Development

To run the project locally:

```bash
npm install
npm run dev
```

## Tech Stack

- React
- Vite
- Firebase
- CSS
